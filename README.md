# 📘 Read Me – Excel Data Analysis Project

## 📌 About the Project

This project focuses on analyzing employee data using Microsoft Excel. It demonstrates how different Excel functions can be applied to perform calculations, data cleaning, and decision-making tasks in a real-world dataset.

---

## 🎯 Objective

The main objective of this project is to:

* Understand and apply core Excel functions
* Perform data analysis on employee records
* Clean and transform raw data into useful insights
* Build a strong foundation for data analyst roles

---

## 🛠️ Functions Used

This project covers the following Excel functions:

* **SUM / SUMIF** → Salary calculations
* **AVERAGE / AVERAGEIF** → Mean values analysis
* **COUNT / COUNTIF** → Data counting and filtering
* **IF** → Logical decision making
* **MAX / MIN** → Finding extreme values
* **VLOOKUP** → Data retrieval
* **CONCAT / TEXTJOIN** → Data combining
* **LEN** → Text length analysis
* **TRIM** → Data cleaning (removing extra spaces)

---

## 📊 Key Tasks Performed

* Total and conditional salary calculations
* Employee performance classification
* Department-wise and city-wise analysis
* Lookup operations using Employee ID
* Text cleaning and formatting
* Identifying duplicate or extra spaces in data

---

## 💡 Learning Outcome

After completing this project, I learned:

* Practical use of Excel in data analysis
* How to clean messy datasets
* How to apply formulas efficiently
* How to solve real-world business problems using Excel

---

## 🚀 Conclusion

This project enhances practical Excel skills and provides hands-on experience required for entry-level roles like **Data Analyst**.
